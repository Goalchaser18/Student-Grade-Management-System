# Student Grade Management System

A console based C++ application to manage Student Grades

## Features
- Add Student details including Name, Roll no. and marks of 5 subjects
- View report of individual student by entering the serial number
- Calculate total marks,percentage and grade automatically
- Find the topper among all students

## How to run
`g++ Student_Grade.cpp -o Student_Grade`
`./Student_Grade`

## Skills used
C++, Stucts, Arrays, Functions, Loops
